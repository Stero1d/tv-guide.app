Для того чтобы запустить теле гид, необходимо:
1) Скачать и установить один из существующих локальных серверов(winginx(запустить, на панели задач открыть его и нажать запустить все), denver и т.д.);
2) Закинуть содержимое папки tv-gid в localhost локального сервера(Например: winginx\Home\localhost\public_html\(содержимое папки);
3) В адресной строке браузера ввести localhost;

В приложении реализованы следующие пункты:
1) Вывод текущей даты, дня недели и времени;
2) Отображение временой шкалы около 1 часа, с обновлением в 30 мин интервала;
3) Передвижение слайдера(текущей временной линии в пределах 30 мин.
4) Получение из модели данных 2 JSON файлов (channels,programs), обработка и объединение их по ид_канала = ид_программ, каналы также идут по порядку lcn;
5) Вывод их с помощью шаблона на страницу;
6) Переключение страниц каналов(круговое);
7) Показа описания программы по нажатию на ее блок с названием и отображении описания канала при нажатии на блок с названием канала;
